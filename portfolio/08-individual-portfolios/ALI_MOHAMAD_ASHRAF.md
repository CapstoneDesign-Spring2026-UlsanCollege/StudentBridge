# Challenges Faced and Solutions

### Challenge 1: Authentication Failures

Users could not access the platform if database connectivity was unavailable or if account records were missing.

**Solution:**

* Verified MySQL database configuration.
* Created and maintained test user accounts.
* Conducted login testing using multiple user scenarios.

### Challenge 2: Frontend–Backend Synchronization

Ensuring that user actions on web pages correctly triggered backend operations required careful testing.

**Solution:**

* Validated servlet mappings.
* Tested request parameters and responses.
* Fixed navigation and redirection issues during integration.
